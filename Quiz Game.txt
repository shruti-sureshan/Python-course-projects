This QUIZ GAME is a python application, specially designed for the purpose of taking tests. The main objective of our project is to efficiently evaluate the candidate thoroughly through a fully automated system .It saves time as it allows students to give test without the need of extra materials like pen, paper etc and displays the results as the test gets over, so no need to wait for the result.
The purpose of this project is to build an application program to reduce the manual work for managing the questions, answers, calculating score. This quiz game is designed to increase learning of players by playing a multiple choice quiz contest on various topics which is interactive, user friendly and fun to play.
